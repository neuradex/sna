# hello-sna

Minimal SNA sample — the full skill -> event -> GUI loop in one project.

## Quick Start

```bash
cd sna/samples/hello-sna
npm install
npx sna up
```

Click "Say Hello" in the browser. Watch events flow from Claude Code to the GUI in real time.

## What's inside

- **`.claude/skills/hello/SKILL.md`** — A skill that greets you by name
- **`src/App.tsx`** — React UI with typed skill invocation and real-time event display
- **`src/sna-client.ts`** — Type-safe skill client (generated by `sna gen client`)

## The SNA Loop

```
Button click -> skills.hello({ name: "World" })
  -> SDK spawns Claude Code session
  -> Claude reads SKILL.md, runs emit.js
  -> Events stream via SSE
  -> useSkillEvents updates the UI
```

## Regenerate the client

After editing `sna.args` in `SKILL.md`, regenerate the typed client:

```bash
npx sna gen client
```
